removeMatching(obj)

Removes all elements that match the specified object or function.
Parameters:
Name Type Description
obj Can be any javascript type or a can be a function that recieves a list element and returns a boolean. If a function is supplied, the function will be called on all items and items for which the function returns true will be removed. If the parameter is not a function, a deep equality comparison will be performed to remove items.
Source:
  • LinkedList.js, line 155
Examples
//The following snippet will remove all even numbers
//from a linked list of numbers.
list.removeMatching(function (index, item) { return item % 2 == 0; });
//The following snippet will remove all nodes storing 
//the specified object using a deep equality comparison.
list.removeMatching({ library: 'jstructs', author: 'Mehdi' });
//The following snippet will remove all nodes storing 
//the specified string.
list.removeMatching('Hello World');
